# KSchrome
![image1](sampleimage/image1.png "image1")  
強震モニタ＆地震情報表示ソフト
# 機能
* 強震モニタの表示
* Wolfx APIから取得した緊急地震速報の表示
* Yahoo地震から取得した地震情報の表示
* 最寄りの観測点の震度・加速度の表示
# インストール方法
1.ブラウザでchrome://extensions/を開く  
2.デベロッパーモードをONにする  
3.左上の"パッケージ化されてない拡張機能を開く"をクリックしてダウンロードして解凍したファイルを開く  
4.ブラウザ画面右上の拡張機能アイコンをクリックしてKSchromeをクリックして開く
